# MSP Central UI Clone (Next.js + Tailwind)

Pixel-close UI clone of the ManageEngine “MSP Central” marketing pages based on provided screenshots (UI-only; placeholder content/images).

## How to run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/` (Overview/Home)
- `/features`
- `/resources`
- `/support`
- `/get-quote`

Placeholder routes also exist for items shown in the header/top bar (e.g. `/demo`, `/solutions`, `/free-trial`).
